# Research Track

This folder is the index for the extensible, research-oriented pipeline.

Keep the more advanced RC variants, diagnostics, and campaign automation here or referenced from here.

Suggested reference points:

- `mpc/` for the MPC execution layer and predictor implementations
- `pinn_model/` for the PINN training and architecture code
- `configs/` for experiment variants and future scaling studies
- `plans/` for campaign and execution planning
- `methodology/` for training and evaluation methodology notes
- `diagnostics/` for validation and troubleshooting guides
- `summaries/` for coverage and final project summaries
- `notes/` for ad hoc reviews and index notes
